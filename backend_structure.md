# Backend

## Directory Structure

- `controllers/`         # Handles the logic for the routes
- `models/`              # Contains the data models
- `routes/`              # Defines the different routes for the application
- `middlewares/`         # Contains middleware functions
- `services/`            # Contains business logic and services
- `config/`              # Holds configuration files such as database config

## Main Files

### server.js
This is the main file that initializes the server.

### package.json
This file defines the dependencies and scripts for the Node.js application.
